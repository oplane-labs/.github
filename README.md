# Oplane Labs

Oplane Labs is the public home for demo repositories, reference implementations, and partner-facing examples from Oplane.

Oplane helps development teams identify and address security risks at the architecture and design level. Instead of only scanning for code-level vulnerabilities, Oplane analyzes how systems are intended to work, how data flows through them, and where abuse cases or missing security requirements may appear.

This organization is where we publish code that is useful to share outside the core product repositories.

## What You Will Find Here

- Demo applications that show Oplane threat modeling workflows in realistic codebases
- Example integrations for GitHub, GitLab, MCP, and developer environments
- Public reference code for partners, customers, and the security community
- Small utilities or templates that support secure development workflows

## About Oplane

Oplane brings threat modeling into the development loop:

- Use Oplane locally through MCP-compatible tools while building features
- Run automated threat modeling on pull requests and merge requests
- Generate specific security requirements with implementation guidance
- Track requirement status, risk decisions, and security posture over time

Learn more in the [Oplane documentation](https://oplane.io/docs).
